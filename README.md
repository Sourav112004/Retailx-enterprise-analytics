## Project Overview

RetailX Enterprise Analytics is an end-to-end retail analytics project designed to simulate a real-world analytics workflow.

The project demonstrates:

* Relational Database Design
* Data Profiling using Pandas
* Data Cleaning
* SQL Business Analysis
* Interactive Power BI Dashboards
* Project Documentation
* Version Control using Git

## Sales Analysis

### Revenue by Product Category
- Calculated total revenue for each product category.
- Identified the Top 10 revenue-generating product categories.

### Customer Revenue Analysis
- Calculated total spending for each customer.
- Identified Top 10 customers by revenue.

### Seller Performance Analysis
- Calculated revenue generated by each seller.
- Identified Top 10 revenue-generating sellers.

### Product Pricing Analysis
- Calculated average selling price by product category.
- Ranked product categories based on average selling price.

### Customer Purchase Behaviour
- Calculated the number of distinct product categories purchased by each customer.
- Explored customer purchasing diversity using `COUNT(DISTINCT)`.

## Key Learning Outcomes

- Identifying the correct grain before writing SQL.
- Determining when joins are necessary and avoiding unnecessary joins.
- Translating business questions into SQL queries.
- Selecting appropriate aggregate functions based on business requirements.
- Understanding the business value behind analytical queries.
- Presenting insights from a business perspective rather than focusing solely on SQL syntax.
